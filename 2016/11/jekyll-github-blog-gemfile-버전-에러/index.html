  2. bundle install

오오…역시 스택오버플로우 짱. 루비도 한번 제대로 공부해보고 싶다. 항상 오류가 날때마다 검색후 오류를 해결하긴 하지만 왜 오류가 났는지, 왜 오류가 해결되었는지에 대한 배움이 전혀 없다.
